# Register API

## Description

I recommend installing MySQL Workbench 8.0.33.

## Documentation

Featured by Open API specification (Swagger) you can see the documentation accessing /docs endpoint.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
